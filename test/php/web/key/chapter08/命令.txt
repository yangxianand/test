8.3.1 数据备份

在命令行中切换到C:\web\mysql5.7\bin
# ① 备份mydb数据库的student表
mysqldump -uroot -p mydb student > D:/student.sql
# ② 备份单个数据库
mysqldump -uroot -p mydb > D:/mydb.sql
# ③ 备份所有数据库
mysqldump -uroot -p --all-databases > D:/mysql.sql

8.3.2 数据还原

1.输入重定向
# ① 登录MySQL，创建mydb2数据库后退出
# ② 还原数据
mysql -uroot -p mydb2 < D:/mydb.sql
