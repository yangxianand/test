5.2.1 安装MySQL

安装MySQL：
cd C:\web\mysql5.7\bin
mysqld -install mysql5.7

卸载MySQL：
mysqld -remove mysql5.7

初始化MySQL：
mysqld --initialize-insecure

启动或停止MySQL：
net start mysql5.7
net stop mysql5.7

5.2.2 登录MySQL

启动MySQL命令行工具并登录：
cd C:\web\mysql5.7\bin
mysql -u root

设置root用户的密码：
ALTER USER 'root'@'localhost' IDENTIFIED BY '123456';

有密码的用户登录：
mysql -uroot -p123456

查看数据库：
SHOW DATABASES;
